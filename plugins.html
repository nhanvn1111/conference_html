Pe Slider

Include the plugin file: 

<script src="pe-slider.min.js">

To create a slider use a markup with this structure: 

<!-- A wrapper that can be targeted during the init() -->
<div id="demo-slider">
  <!-- The Slides are inside an unordered list <ul> with each slide a list item <li>. No limit in the number of slides and content can be an <img>, <video> or any HTML content -->
  <ul class="pe-slides">
   <li class="selected">
     <img class="full-slide" src="first-slide.jpg" />
   </li>
   <li>
     <img class="full-slide" src="second-slide.jpg" />
   </li>
   <li>
     <img class="full-slide" src="third-slide.jpg" />
   </li>
 </ul>
 <!-- You can add a navigation (Edit with CSS if needed) -->
 <ul class="pe-slider-nav">
   <li><a href="#0" class="pe-slider-prev"></a></li>
   <li><a href="#0" class="pe-slider-next"></a></li>
 </ul><!-- /.pe-slider-nav -->
</div><!-- /.pe-slider-wrapper -->

Then initialize and/or customize it: 

// Initialize the slider
var mySlider = new peSlider();
mySlider.init();
// Or, in case you want to manipulate some options...
mySlider.init({
 slider: '#my-slider', 
 navigation: false,
 pagination: true,
 auto: true,
 delay: 5000,
 hoverPause: true
});

Options

slider is the selector for the slider.
It could be a css selector like '#my-slider' or an Element Object like document.getElementById('my-slider') depending on your needs.
Default: '.pe-slider'

navigation show/hide the arrows.
It should be a Boolean like true or false
Default: true

pagination show/hide the pagination dots.
It should be a Boolean like true or false
Default: true

auto autoplay slides.
It should be a Boolean like true or false
Default: false

delay is the time between slides when auto is enabled.
It should be a Number like 3000
Default: 5000

hoverPause allows to stop the autoplay when the mouse is hovering the slider and resume when the mouse is outside.
It should be a Boolean like true or false
Default: false

hoverNav only shows the navigation arrows when the mouse is hovering the slider container.
It should be a Boolean like true or false
Default: false

infinite allows the navigation to loop the slides.
It should be a Boolean like true or false
Default: true

carousel changes the behavior from slider to carousel.
It should be a Boolean like true or false
Default: false

overflow shows the content outside the slider container. Best with carousel carousel active.
It should be a Boolean like true or false
Default: false

swipe adds the ability to control the slider by dragging/swiping it.
It should be a Boolean like true or false
Default: false

touch enables the swipe function only for mobile touch devices
It should be a Boolean like true or false
Default: true

beforeChange is a function fired just before each slider changes.
It should be a Function like function(){ alert('just before change!') } or function(){ mySlider.classList.add('changing') }
Default: null

afterChange is a function fired just after each slider changes.
It should be a Function like function(){ alert('just after change!') } or function(){ mySlider.classList.remove('changing') }
Default: null

Functions

activeSlide() is a function that returns the active slide when it is called.
Could return the number of the node element of the DOM.

// will return the number of the active slide, zero based.
mySlider.activeSlide().number /* for example 2 */ 
// will return the DOM element of the active slide
mySlider.activeSlide().elem /* for example <li id="s2" class="slide-2 selected">...</li> */

totalSlides() is a function that returns the total number of slides.

mySlider.totalSlides() /* for example 3 */

nextSlide() and prevSlide() are functions that trigger the slider to move to the next/previous slide. It allows you to fire those actions from other DOM elements.

$('#myButton').click(function(){
  mySlider.nextSlide();
})
<button onclick="mySlider.prevSlide()">Next slide</buttton>

getSlider() is a function that could return the slider element, the wrapper element and the slides NodeList. 

var sliderStuff = mySlider.getSlider(); /* here is the Object */
var wrapper = sliderStuff.wrapper;
var slider = sliderStuff.slider;
var slides = sliderStuff.slides;
// Examples
wrapper.style.height = '340px'; /* set the height of the wrapper */
slider.addEventListener('click', myFunction); /* add an event when the slider is clicked */
console.log(slides[0].id) /* get the ID of the first slide */

You can also change the slider transition effect. Adding the class
pe-slider__scale or pe-slider__flip to the <ul class="pe-slider">...</ul>

Pe Navigation Scroll

We use this plugin for the main navigation in all of the Ebook pages.
It animates a page scroll to the various navigation links anchors and adds a class to the current link according to the scroll position.

Include the plugin file: 

<script src="pe-navscroll.min.js">

Use a markup like this:

<nav class="navbar">
  <ul class="nav nav-pills">
   <li class="active"><a href="#section-one">Section 1</a></li>
   <li><a href="#section-two">Section 2</a></li>
   <li><a href="#section-three">Section 3</a></li>
 </ul>
</nav><!-- /.navbar -->

Then initialize and/or customize it:

// Initialize the default navScroll
peNavScroll.init();
// Or, in case you want to manipulate some options...
peNavScroll.init({
  nav:'.navbar',
  class:'active',
  duration: 500,
  hashChange: false,
  offset: 40,
  allLinks: true, // Allows the page scroll animation to be triggered from anchor links outside the navigation "nav" element
  easingFunction: easing.easeInOutCubic // the easing function
});

Options

nav is the selector for the navigation.
It should be a css selector like '#navigation' or ul.main-navigation.
Default: '.navbar'

class is the class to add to the navigation item, when the scroll position corresponds to the item entering the viewport.
It should be a String for the class name like 'current'.
Default: 'active'

duration is the duration of the animated scroll.
It should be a number like 500.
Default: 900

hashChange adds the anchor (#hash) of the item clicked at the end of the url.
It should be a Boolean like true or false.
Default: true

offset modifies when the active class is added by a number of pixels to get a finer control on the class change trigger.
It should be a number like: 60 or -40.
Default: 0

allLinks allows the page scroll animation to be triggered from anchor links outside the navigation "nav" element.
It should be a Boolean like true or false.
Default: false

noMenu use in cases when the menu height should not be a factor to calculate the scroll (floating menu: absolute/fixed).
It should be a Boolean like true or false.
Default: false

hideOnScroll adds the class nav-hide when the user scrolls down and nav-show when the user scrolls up and also at the beginning and end of the current page.
It should be a Boolean like true or false.
Default: false

hideFn is an additional function function called when the user scrolls down. Works only when hideOnScroll active.
It should be a Function like function(){ $('.nav').hide(); } or function(){ console.log('hide!') }.
Default: null

showFn is an aditional function called when the user scrolls up. Works only when hideOnScroll active.
It should be a Function like function(){ $('.nav').show(); } or function(){ console.log('show!') }.
Default: null

easingFunction is the easing function for the animated scroll.
It should be a function like easing.easeInOutCubic.
Default: easing.linear

Available easing functions: easing.linear, easing.easeInQuad, easing.easeOutQuad, easing.easeInOutQuad, easing.easeInCubic, easing.easeOutCubic, easing.easeInOutCubic, easing.easeInQuart, easing.easeOutQuart, easing.easeInOutQuart, easing.easeInQuint, easing.easeOutQuint, easing.easeInOutQuint.

Pe Scroll Change

This plugin lets you add or remove a CSS class from a DOM element when a user scrolls the document.

Include the plugin file: 

<script src="pe-scrollchange.min.js">

Then initialize it and/or customize it: 

// Initialize the default scrollChange
peScrollChange.init();
// Or, in case you want to manipulate some options...
peScrollChange.init({
  elem: '.navigation',
  trigger: '#trigger',
  classesToChange: 'fixed',
  offset: 0,
  removeClass: false,
  endPoint:triggerH
})

Options

elem is the element to control.
It could be a css selector like '#navigation' or an Element Object like document.getElementById('navigation') depending on your needs.
Default: '.navbar'

trigger is the point where the action (change of the class) will occur.
It could be a css selector like '#trigger', an Element Object like document.getElementById('trigger') or a Number like 50. If it is a number, the action will occur after the quantity of pixels specified has been scrolled. If there is a DOM element (through a css selector or an Element Object) instead, the trigger will occur when the page scrolls where the element is on the page Y coordoninate.
Default: 50

classesToChange are the classes to add to or remove from the element.
It should be a String with one or more classes separated by a space i.e: 'fixed transparent' or fixed-nav.
Default: nav-bg

offset is the quantity of pixels to sum or rest from the trigger point to finetune when the action occurs.
It should be a Number like: 100 or -40.
Default: 0

removeClass changes the default logic and removes the class or classes instead of adding them when the page scrolls to the trigger point.
It should be a Boolaen like: true or false.
Default: false

endPoint creates an end point for the action. Instead of adding/removing class/es after the trigger point, it allows to add/remove the classs/es between the trigger point and the endPoint.
It should be a number like : 700 or trigger.offsetHeight.
Default: 0

Pe Bg

This plugin lets you load dynamically a background image to a html tag such as a section and add extra effects like parallax.

Include the plugin file: 

<script src="pe-bg.min.js">

Then initialize it and/or customize it: 

// Initialize the default Bg
var myBgs = new peBg();
myBgs.init();
// Or, in case you want to manipulate some options...
myBgs.init({
  elems: '.section-background',
  bgSrc: 'assets/img/bg_1.jpg',
  bgColor: '#e04863',
  blend: 'multiply',
  bgOpacity: .3,
  parallax: true,
  factor: 0.55,
  extraHeight: 'x2'
})

Options

elems are the elements that will have the ability to load a Background.
It could be a css selector like '.section-background' or an Array with specific css selectors like ['.my-section', '#bg-1', '.another-bg'] depending on your needs.
Default: '.section-background'

bgSrc is the url for the background image.
It should be a String url like 'img/my-bg.jpg'
Default: ''
You could also control this option adding for example data-bg-src='img/my-bg.jpg' to the DOM element

bgColor is the color of the background overlay.
It should be an html color like '#e04863' or 'white' or 'rgba(25,67,34,.5)'
Default: ''
You could also control this option adding for example data-bg-color='#e04863' to the DOM element

blend is the blend mode to use to mix the background overlay color with the picture.
It should be a String of one of the background-blend-mode's like: 'multiply', 'screen', 'overlay', 'darken', 'lighten', 'color-dodge', 'color-burn', 'hard-light', 'soft-light', 'difference', 'exclusion', 'hue', 'saturation', 'color' or'luminosity'
Default: 'normal'
You could also control this option adding for example data-bg-blend="multiply" to the DOM element

parallax moves the background relatively to the page scroll movements.
It should be a Boolean like true or false
Default: false
You could also control this option adding the class pe-parallax to the DOM element

factor is the 'magic number' to control the intensity of the parallax.
It should be a float Number between 0 and 1 like 0.5 or 0.76
Default: 0.55
only works with parallax="true"

bgOpacity controls the opacity of the parallax background.
It should be a float Number between 0 and 1 like 0.3 or .85
Default: 1
You could also control this option adding for example data-bg-opacity='0.3' to the DOM element
only works with parallax="true"

mobile enable/disable the parallax behavior for mobile devices.
It should be a Boolean like true or false
Default: true
only works with parallax="true"

extraHeight multiplies the height of the parallax image x times (useful for large sections).
It should be a String like 'x2' or 'x3' or 'x4'
Default: false
You could also control this option adding the classes x2, x3 or x4 to the DOM element
only works with parallax="true"

fixed the background will remain attached when the page scrolls.
It should be a Boolean like true or false
Default: false
You could also control this option adding the class pe-fixed to the DOM element
only works with parallax="false"
Fixed background option has been disabled by default on mobile browsers because of important performance issues.
