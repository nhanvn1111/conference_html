Checkers

The Checkers work as a pair of content.
The second item will inherit a background-color on :hover by adding a color class such as chk-colorname to the chk-container.

You can create a featured checker with a background-color by adding the class active to the chk-container.

The checkers are distributed in equal width for each row.

You need one chk-container div for each row and then a new chk-container div for each new pair.

Font size & Line height

You can easily change the font-size and the line-height with our specific multiplicator classes that will multiply the element inherited font-size/line-height. Those classes use em as a multiplier class. For example: adding em-1-5 to an element with a parent font-size:10px will result in a 15px font size (10px * 1.5em = 15px).

Below the classes for font-size

em-0-25 - em-0-5 - em-0-75 - em-1-25 - em-1-5 - em-1-75 - em-2 - em-2-5 - em-3 - em-3-5 - em-4

Below the classes for line-height

lh-0-25 - lh-0-5 - lh-0-75 - lh-1-25 - lh-1-5 - lh-1-75 - lh-2 - lh-2-5 - lh-3 - lh-3-5 - lh-4

Here is an example <p class="em-1-75">Lorem ipsum (18px * 1.75em = 31.5px)</p>

Lorem ipsum (18px * 1.75em = 31.5px)

Here is another example <p class="lh-0-75">Lorem ipsum... (18px * 0.75em = 13.5px)</p>

You can also use some more specific font size helpers like for instance the class fz-18 wich equals to font-size:18px . As default we included a list of fz-* classes from 10 to 20 in incremental steps of 1px. This way you can mix a fz-* class and a em-* class to set almost any font-size/line-height for any specific element. If needed you can change the mixins in our _modules.scss file.

RHYTHM SPACING

We have created a list of spacing class helpers to override any element general spacing while maintaining the rythm. For instance you can use those classes: mb1, mb2, mb3, mb4, mb5 to add a margin-bottom to an element.

To respect the spacing rythm those helper classes are based on thehtml { font-size: ... } value. For example, the classes that ends with:
1 are 0.5rem relative to the element html font-size
2 are 1rem relative to the element html font-size
3 are 2rem relative to the element html font-size
4 are 4rem relative to the element html font-size
5 are 8rem relative to the element html font-size

There is a sass @mixin to generate this relation which you can find in the _modules.scss file.

SPECIFIC SPACING IN PIXELS

We also have a list of spacing helpers that allow you to control the spacing with pixel. For example, the class mb-15 adds a margin-bottom: 15px to the elememt. (Note that mt-* is px based and mt* is rem based)

As a default there are classes to control margin and padding from 5px to 25px in incremental steps of 5px. For example: mt-5, mt-10, mt-15, mt-20, mt-25.

As usual you can edit and expand those classes with our @mixin in the _modules.scss file.

Adding your own styles

If you need to override, add or change classes for your project please write them in the custom.css file provided and link it to your project. This will allow you to update to future versions without loosing your changes.
